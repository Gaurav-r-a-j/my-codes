Windows 10/11 Shortcuts

  1. Minimize windows and open desktop = WIN + D
  2. Get snipping tool = WIN + SHIFT + S
  3. Get recent closed tabs back = CTRL +SHIFT + T
  4. Open settings = WIN + I
  5. Create New virtual desktop =WIN + CTRL + D
  6. To move between virtual desktop = WIN + CTRL + D
  7. For quick copy pasting = CTRL + (drag and drop the file)
  8. Open quick settings during game = WIN + G
  9. Open file explrer = WIN + E
  10. Activate Windows AI = WIN + C
  11. Open Task Manager = CTRL + SHIFT + ESC
  12. Open apps from taskbar =WIN + (App No.)
  13. To get all open tabs = CTRL+ALT+TAB
  14. Quick Lock =WIN + L
  15. Rename file = select file and press F2
